You are a senior full-stack engineer and product designer.
Build a production-ready agricultural marketplace platform that connects verified farmers directly to bulk buyers, eliminating middlemen.
The platform must work seamlessly on mobile and desktop (responsive first) and prioritize a modern, premium UI/UX.

Core Product Description

Create a digital agricultural exchange where farmers can list produce and livestock (live or processed) and bulk buyers can discover, bid, contract, and purchase securely.

The app must support:

Produce (vegetables, grains, fruits)

Livestock (chickens, fish, goats, pigs – live or dressed)

Transport & logistics coordination

Secure escrow payments

Reviews, ratings, and verification

Tech Stack Requirements

Frontend: React / Next.js (responsive, mobile-first)

Styling: Tailwind CSS + modern design system

Backend: Node.js (Express or Next API routes)

Database: PostgreSQL (preferred) or MongoDB

Auth: Role-based authentication (JWT / OAuth)

Payments: Escrow-ready payment architecture (abstracted, provider-agnostic)

Deployment-ready: Clean env variables, scalable structure
